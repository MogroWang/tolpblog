TOLP 4.0.001 DEV 更新日志

发布于 2026-01-22 01:56:31

《光点之旅》第四次开发计划 虽迟但到！

首先很抱歉又咕咕咕这么久，但是也很高兴本项目继续到了今天。

目前本项目已有些许灵感，正在缓步恢复更新节奏中。

后续更多更新日志、人物设定、玩法设计等将在《光点之旅》官方网站、公众号、工作室B站账号等平台同步发布。

我也正在学习与探索更多内容的开发，敬请期待。

关于本次更新，先从UI着手。

由于目前拟物化重新开始流行，本项目也决定将扁平化和拟物化融合起来设计UI。

当然《光点之旅》的整体风格将不会有太大变化。

期待在下次更新见面哦！(´･ω･)ﾉ

⚠️ 本次更新仅在内部构建，不对外开放测试。

主要更新

UI

  • 更新 MogroWang Studio LOGO 至 11.1 版本
  • 更新 MogroWang Studio LOGO 启动界面音乐（更换更轻柔的合成器音色，原贝斯换为吉他）
  • 调整 游戏开头工作室 LOGO 的缩放补间大小 以适应新版 LOGO 的替换
  • 更新 TOLP LOGO 至 3.0 版本
  • 调整 TOLP LOGO 的大小 以适应新版 LOGO 的替换
  • 更新 主菜单按钮贴图
  • 更新 主菜单按钮音效（更换更轻柔的合成器音色）
  • 调整 主菜单按钮动效（鼠标悬停时会变化为不同颜色）
  • 新增 主菜单按钮旋转动画
  • 新增 按钮点击时会有不同音调的音效
  • 新增 光标在主菜单按钮上悬停时 对按钮的提示与介绍
  • 更新 关于页（排版布局重构，内容有所更改）
  • 更新 设置页（排版布局重构，设置项目不变）
  • 更新 存档页（新增传送门粒子）

错误修复

  • 修复了 部分场景中鼠标悬停在按钮上时 按钮颜色变化报错 的错误
  • 修复了 主菜单 转到 存档页后音效立刻停止 的错误

保留特性

  • （仅PC）当鼠标悬停在主菜单按钮的边缘时，会断断续续触发按钮的悬停动画，配合音效产生节奏。（原理：按钮贴图为方形，旋转时碰撞遮罩也随之旋转。）

计划处理

  • 修复更多场景中 鼠标悬停在按钮上时 按钮颜色变化报错 的错误
  • 存档BUG：在2号档位进行删除操作时 再删除第二个存档 会出现未完全删除的严重错误？
  • 当主菜单在跳转其他场景的补间时，快速点击主菜单按钮，再返回主菜单，会出现反复跳转的情况。
  • 添加音量等相关设置。
  • 考虑移动端的性能，可添加控制粒子特效质量的设置。
